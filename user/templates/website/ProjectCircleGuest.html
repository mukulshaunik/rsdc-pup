What would I do?

As a member of the Project Circle, you would assist with a range of tasks including:

  • Identification of Project problem
  • Explaining specific project problems
  • Literature review
  • Specifying the purpose of project
  • Specification of a Conceptual framework - Usually a set of hypotheses
  • Choice of a methodology (for data collection)
  • Data collection
  • Analyzing and interpreting the data
  • Developing Project
  • Communicating the Project findings and, possibly, recommendations

How to start a new Project?

To start a new Project , firstly you have to register yourself with us and then after login you'll see a add Project tab on this page from where you can add a new Project by giving some details about the Project you want to start or you have already started.

How to be a part of someone's Project?

To be a part of someone's Project, firstly you have to register yourself with us and then after login you can click on the Intern circle tab and select someone's Project and apply for it by giving some details like why do you want to join?, Any Project experience? Etc. If the Leader head accepted your request then you can be the part of that Project.
